# 🔋 Battery Cell Energy Predictor

This Gradio-based web app predicts **energy consumption** in a battery cell manufacturing batch based on:

- 🧱 Batch Size (100–500)
- 🌡 Temperature (20–40°C)
- ⚙️ Machine Load (0.5–1.5)
